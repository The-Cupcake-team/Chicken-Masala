# Chicken-Masala

# Introduction
An Indian food app is designed to help people discover Indian cuisine.With this app, users can browse through a vast selection of Indian dishes.
The app provides detailed descriptions and photos of each dish, as well as information on the ingredients and cooking methods used.
Overall, an Indian food app is a convenient and user-friendly way to enjoy the delicious flavors of Indian cuisine.
